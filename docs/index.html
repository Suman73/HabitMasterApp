HabitMaster

Minimal habit tracker. Use the web app (with Firebase login & sync) or the Edge extension (local-only, no login).

Edge Extension

Runs entirely in the popup using chrome.storage.local — no network calls.
Download (.zip)

Install in Microsoft Edge

  1. Open edge://extensions
  2. Enable “Developer mode”
  3. Click “Load unpacked” and select the extracted edge-extension folder
  4. Pin the extension and click the icon to open the popup

Web App (with Login)

Hosted static SPA with Firebase Auth (Google/Anonymous) and Firestore sync.
Open Web App

If sign‑in fails, add your Pages domain (e.g., YOUR-USER.github.io) to Firebase Auth → Authorized domains.

What’s inside

  • Edge extension: manifest.json, popup.html, popup.js, background.js, storage.js, date-utils.js, styles.css
  • Web app: index.html, app.js, date-utils.js, styles.css, firebase-config.js
